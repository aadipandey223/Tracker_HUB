## ğŸš€ Getting Started

### Prerequisites
- Node.js installed on your machine.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/tracker-hub.git
    cd tracker-hub
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  **Open in Browser**
    Visit `http://localhost:5173` to start using Tracker Hub!

## ğŸ”’ Privacy & Data

Tracker Hub is designed to be **Local Only** by default. We removed all server-side authentication dependencies. 
-   Your data is stored in `localStorage` in your browser.
-   Clearing your browser cache/data will remove your Tracker Hub data.
-   We recommend using the **Export** feature in the Finance tab regularly to back up your critical data.
